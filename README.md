# 中国农大Urp绩点计算
作者：张云浩    QQ : 3358023393(新)



## **如何使用**

- 点击上方Download ZIP按钮，即可将整个项目下载至电脑。
- 解压ZIP文件，打开其中/User目录，点击'urp_gpa.exe'文件，输入Urp教务系统账号和密码即可。

**注**：

- 建议使用 管理员模式 + win7兼容模式 打开(为了最大的兼容性,打包环境为win7-32bit)。
